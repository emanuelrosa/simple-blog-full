Modelo de blog simples com área administrativa para gerenciamento das postagens
feito em Php + Mysql, com PDO.

Informações gerais: 
    Area adminstrativa:
        - Cadastro de categorias
        - Cadastro de autores 
        - Envio de postagens com imagem escolhendo o autor para a postagem
        - Usuários do sistema administrativo
        - Configurações de informações do blog
            - Titulo
            - Sescrição
            - Palavras chaves
            - Autor

        - Integrações
            - Header e Body

        - Redes sociais para compartilhamento
            - Facebook
            - Instagram
            - Twitter
            - Whatzapp
            - Google+
            - Pinterest

        - Configurações de imagens do blog
            - Topo
            - Logo
            - Painel administrativo
            - Banner de divulgação (para compartilhamento do blog)
            - Gif de preload

        - Institucional
            - Sobre o blog
            - Termos de uso
            - politica de privacidade
            - Mensagens de erros (recebidas pelo blog)
            - Mensagens do site

    Blog:
        - Layout simples com imagem no topo, menu de categorias, coluna direita com as postagens
mostrando imagens e resumo, coluna esquerda contendo área de pesquisa,últimas postagens, melhores
da semana.
